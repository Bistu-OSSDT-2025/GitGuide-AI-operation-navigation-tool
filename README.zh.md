# GitGuide-AI 操作导航工具

> ? 查看英文版：[English README](./README.md)

## ? 项目简介

GitGuide-AI 操作导航工具是一个智能助手，用户可以通过自然语言指令跳转网页页面。比如在一个有 1、2、3、4 四个板块的网站中，输入“打开第 3 部分”，网页即可自动跳转到相应位置。

该工具特别适合访问结构清晰网页的用户，通过自动化简化重复操作流程。

## ?? 核心功能

- ? **自然语言跳转指令**：输入文字即可跳转网页特定部分。
- ? **网页结构分析**：分析提供的网页源代码（HTML DOM）。
- ? **Python 自动化脚本**：根据分析结果生成跳转、点击等脚本。
- ? **用户常用网页存储**：保存常用网页，便于快速复用自动化脚本。

## ? 工作流程

1. 用户提供网页（需有源代码访问权限）
2. 工具分析页面结构
3. 用户输入跳转指令（如：“跳转到第3部分”）
4. 自动执行对应 Python 脚本完成跳转或操作

## ? 项目目标

在具备源码支持的前提下，帮助用户快速完成网页跳转和常用操作，简化操作流程，提高效率。

## ? 技术栈

- Python
- 网页结构分析（如 BeautifulSoup, lxml）
- 自动化操作（如 Selenium, Playwright）
